# Machi Koro – Game Notes (Structured)

## 👥 Players
- Each player is a mayor  
- 2–4 players  

---

## 🏁 Game Goal
- Build the city the quickest and win  
- Unlock all landmarks → first player wins  

---

## 🏙️ Starting State

- Each player starts with:
  - Wheat Field
  - Bakery
  - 4 inactive/gray Landmarks (turned around)
- Each player has:
  - 3 coins (value of 1)

---

## 🏗️ Landmarks (Main 4 Cards)

### Starting Landmarks
- Train Station  
- Shopping Mall  
- Amusement Park  
- Radio Tower  

### Rules
- Start state = 4 landmark cards with gray side facing you  
- Before build → gray card  
- When built → flip → colored card  
- Extra cards go to game box  

---

## 🃏 Establishments (Other Cards)

### Types

- **Green**
  - Give coins only on your turn  

- **Blue**
  - Give coins on your turn and other players’ turns  

- **Red**
  - Take coins from other players  

- **Purple**
  - Player can have only 1 of each type  

---

### Additional Rules

- Player begins with Wheat Field and Bakery  
- Cards are placed face-up in stacks (ordered from small to large)  
- Player can have multiple copies of a card  
  - Cards are stacked so top number is visible  

---

### Activation Order - when same dice number is rolled as on the cards

1. Red  
2. Green  
3. Blue  
4. Purple  

---

## 💰 Coins

- Coin values: 1, 5, 10  
- Each player starts with 3 coins (value 1)  
- Remaining coins are in the bank  

### Usage
- Used to buy buildings  
- Buildings are placed in front of the player  

### Card Effects
- Activated when dice are rolled  
- Effects can give coins from:
  - BANK  
  - OTHER PLAYERS  

---

## 🔄 Turn Logic

- Turn order: clockwise  
- Each turn has 3 phases  

---

## 🎲 Player Turn – Phases

### Phase 1. Roll Dice
- Roll one dice  

---

### Phase 2. Earn Income

1. Check dice value  
2. Check establishment values  
3. If values match:
   - Trigger effect (e.g. get coin from bank)  
4. If not:
   - Nothing happens  

---

### Phase 3. Construction Phase

#### Option A – Buy Establishment

1. Check money amount  
2. Check card availability  
3. Check if money equals card cost  
4. Exchange:
   - money → bank  
   - card → player  
5. Card activates only in future turns (**cheating function possible**)  

---

#### Option B – Build Landmark

1. Pay cost  
2. Money goes to bank  
3. Flip card to colored side  
4. Add additional action  

---

### Additional Rule

- If player owns multiple copies of a card:
  - Effect triggers multiple times  

---

## 🔁 Other Players’ Turn (Your Perspective)

- When opponent rolls dice:
  - Check your cards  

### Effects

- **Blue**
  - Activate → receive coins  

- **Red**
  - Activate → take coins from opponent  

- **Purple**
  - Only activate on your turn  

---

## 💸 Paying Other Players

- If you don’t have enough coins:
  - Pay what you have  

- If multiple players:
  - Pay in counterclockwise order  
  - Continue if coins remain  

---

## 🎯 Starting Player

- Random  
- Dice roll  
- Optional: cheat function (first click)  

---

## 🏁 Game End

- Player wins when all landmarks are built  
- Only first player wins
